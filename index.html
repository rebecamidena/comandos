Comandos de programação

Este site te explicará como funciona os principais comandos de HTML e CSS, que serão: h1, p, img, cor, centralização e tamanho.

O que é o comando "h1"?

O comando "h1" é uma tag do HTML, usado para definir o maior título em um site. No HTML, existe uma hierarquia para classificar os títulos da página que vão do h1(maior e mais importante) até o h6(menor e menos importante).

O que é o comando "p"?

O comando "p" é um comando do HTML que define os parágrafos da página, geralmente, usados embaixo de um título com explicações detalhadas.

O que é o comando "img"?

O comando "img" é usado para adicionar imagens no site para que fique mais atrativo ao olhar do navegador.

Como é feita a formatação de cor?

Para colocar uma cor no CSS, é preciso que declare-a em inglês(exemplo:#FFFFFF que é a cor branca pois representa a intensidade máxima) ou através do RGB indicado através de valores hexagonais(de 0 a 255).

Como é feita a formatação da centralização?

Para centralizar textos de títulos ou parágrafos, é preciso usar a propriedade 'text-align' do CSS da seguinte maneira:

p{ text-align: center }

h2{ text-align: center }

Como é feita a formatação de tamanho?

Para definir um tamanho no CSS, precisa entender que 'width' determina a largura da área do elemento escolhido. E que a área de preenchimento fica dentro da borda e da margem.
